# 国际象棋游戏

一个用React开发的国际象棋游戏，适合前端初学者学习。

## 功能特点

- ✅ 完整的国际象棋规则实现
- ✅ 美观的现代化界面
- ✅ 棋子移动高亮显示
- ✅ 回合制游戏（白棋先手）
- ✅ 简单的游戏状态管理

## 如何运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开显示的地址（通常是 http://localhost:5173）

## 项目结构

```
src/
  ├── components/      # React组件
  │   ├── Board.jsx   # 棋盘组件
  │   └── Square.jsx  # 方格组件
  ├── utils/          # 工具函数
  │   └── chess.js    # 国际象棋逻辑
  ├── App.jsx         # 主应用组件
  └── main.jsx        # 入口文件
```

## 技术栈

- React 18
- Vite (构建工具)
- 纯CSS样式
